# Mirror Simulation Bakeoff

Interactive HTML simulations generated by ChatGPT 5.2, Gemini 3 Pro Preview, and Opus 4.5 for a makeup station mirror configuration project.

This repo accompanies the blog post: [ChatGPT 5.2 vs. Gemini 3 Pro Preview vs. Opus 4.5 -- Mirror Simulator Bakeoff](https://frugal.co/blog/mirror-simulation-bakeoff)

## About

The goal was to create a simulation to validate whether adding pivot mirrors to a makeup station would allow seeing the back of one's head. Each model was given the same prompt and asked to generate a single-file HTML simulation with ray tracing.

Gemini 3 Pro Preview won by producing the only simulation that actually worked for the intended purpose.
